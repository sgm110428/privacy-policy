# FocusGuard 개인정보처리방침

시행일: 2026년 4월 3일

FocusGuard(이하 "앱")는 사용자의 숏폼 콘텐츠 이용 시간을 관리하고, 설정된 기준에 따라 알림 또는 화면 개입 기능을 제공하는 Android 애플리케이션입니다. 본 개인정보처리방침은 앱이 어떤 정보를 처리하는지, 그 정보가 어떤 방식으로 사용되는지를 설명합니다.

## 1. 수집 또는 처리하는 정보

앱은 회원가입, 로그인, 결제, 광고 개인화 기능을 제공하지 않으며, 다음과 같은 정보를 **기기 내부에서만** 처리할 수 있습니다.

- 접근성 서비스 데이터
  - 화면에 표시된 텍스트
  - 콘텐츠 설명(content description)
  - UI 식별자(view id)
  - 현재 실행 중인 앱의 패키지명
  - 현재 화면 문맥(예: YouTube Shorts, Instagram Reels, 댓글 화면, DM 화면 등)
- 앱 설정 정보
  - 사용자가 설정한 제한 시간
  - 언어 설정
  - 서비스 활성화 상태
- 알림 표시를 위한 상태 정보
  - 현재 감지 상태
  - 경과 시간
  - 단계별 개입 상태
- 지원되는 환경에서의 기기 표시 관련 설정 정보
  - 예: 회색조(흑백) 표시 기능 적용 여부

## 2. 정보 처리 목적

앱은 위 정보를 다음 목적에 한하여 처리합니다.

- 숏폼 콘텐츠 화면(예: YouTube Shorts, Instagram Reels, TikTok 등) 감지
- 댓글 화면 등 앱 사용 문맥 구분
- 사용 시간 측정 및 단계별 개입 수행
- 상태 알림, 경고 알림, 화면 오버레이 제공
- 사용자가 설정한 제한 시간 및 언어 설정 저장

## 3. 정보의 전송 및 제3자 제공

앱은 사용자의 개인정보 또는 접근성 데이터를 개발자 서버나 제3자에게 전송하지 않습니다.

- 외부 서버로 전송하지 않음
- 제3자에게 판매 또는 공유하지 않음
- 광고 SDK, 사용자 추적 SDK, 분석 SDK를 사용하지 않음
- 광고 ID를 사용하지 않음

## 4. 정보의 저장 위치 및 보관 기간

- 앱이 처리하는 대부분의 정보는 **사용자의 기기 내부에서만 일시적으로 처리**됩니다.
- 앱 설정(예: 제한 시간, 언어 설정 등)은 기기 내부에 로컬로 저장될 수 있습니다.
- 접근성 데이터를 별도의 외부 서버에 저장하지 않습니다.
- 사용자는 앱 삭제, 앱 데이터 삭제, 접근성 권한 해제 등을 통해 관련 로컬 데이터를 직접 제거할 수 있습니다.

## 5. 권한 사용 안내

앱은 다음 권한 또는 시스템 기능을 사용할 수 있습니다.

### 접근성 서비스
접근성 서비스는 사용자가 현재 숏폼 콘텐츠를 보고 있는지 판단하고, 댓글/DM/일반 화면을 구분하기 위해 사용됩니다. 이 과정에서 화면의 텍스트, 콘텐츠 설명, UI 식별자 등이 기기 내에서 처리될 수 있습니다.

### 알림 권한
앱 상태 알림, 경고 알림, 개입 단계 알림을 표시하기 위해 사용됩니다.

### 표시 설정 관련 기능
지원되는 환경에서는 앱 기능 제공을 위해 회색조(흑백) 표시와 같은 기기 표시 관련 설정을 일시적으로 적용할 수 있습니다. 이러한 처리는 앱 기능 수행 목적에 한정되며, 앱 상태 변경 시 해제되도록 설계되어 있습니다.

## 6. 아동의 개인정보

앱은 별도의 회원가입이나 온라인 데이터 수집 기능을 제공하지 않습니다. 미성년자가 앱을 사용하는 경우 보호자의 지도하에 사용하시기 바랍니다.

## 7. 보안

앱은 처리하는 정보를 외부 서버로 전송하지 않으며, 가능한 범위에서 기기 내부 처리 방식으로 동작하도록 설계되어 있습니다. 다만, 사용자의 기기 보안 상태나 운영체제 환경에 따라 일부 동작 방식에는 차이가 있을 수 있습니다.
